📖 Alfabetización Digital

Fundamentos básicos para el uso correcto de la tecnología

📌 Introducción

La alfabetización digital es el primer paso para desarrollar competencias tecnológicas. Permite que los estudiantes comprendan el uso básico de computadoras, internet, herramientas digitales y seguridad informática.

Este módulo está diseñado para estudiantes que inician desde cero, con el objetivo de fortalecer habilidades fundamentales para el aprendizaje universitario y el uso de plataformas educativas modernas.

🎯 Objetivos del Módulo

  • ✔ Comprender qué es la alfabetización digital.
  • ✔ Identificar partes básicas de una computadora.
  • ✔ Usar sistemas operativos de forma básica.
  • ✔ Manejar archivos y carpetas correctamente.
  • ✔ Utilizar internet de forma segura.
  • ✔ Aplicar herramientas digitales básicas en el estudio.

📚 Contenido del Módulo

💻 1. Introducción a la Computadora

Partes físicas, hardware y software básico.

🪟 2. Sistema Operativo Windows

Uso del escritorio, ventanas y configuración básica.

📁 3. Archivos y Carpetas

Cómo organizar, crear y gestionar información.

🌐 4. Internet Básico

Navegadores, búsquedas y páginas web.

📧 5. Correo Electrónico

Creación y uso de email académico.

🔐 6. Seguridad Digital

Contraseñas seguras y prevención de riesgos.
